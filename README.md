# ArcaeaLib
Useful tools for Arcaea and Phigros
<br/>
## How to use
It's free for you to use it in your project. Just simply type
`
import ArcaeaLib
`
or 
`
import PhigrosLib
`. Remember to add this project name to your project.
## Usage
#### `ArcaeaLib.Aff` for parsing and modifying ArcaeaChartfile.
